User Guide (Click to expand)

1. Create Geometry: Add rectangular blocks by defining their origin (y0, z0) and dimensions (Ly, Lz), as illustrated below.

2. Standard Sections: Use the "Standard Section" button to quickly generate common shapes like I-beams or Z-sections.

3. Snapping: Click "Add by Snapping" then click on any existing corner in the plot to start a new block from that point.

4. Transformed Sections (Rel E):

Use "Rel E" (Relative Stiffness) to model composite materials (e.g. Steel + Aluminum). Define a reference material (Rel E = 1) and scale others relative to it (e.g., Rel E = Emat / Eref). The calculated "Max/Min σxx" will be the actual stress in each material.

5. Stress Analysis: Apply Moments (Myy, Mzz) and Axial Force (Fxx) to calculate stresses. Use the "Stress at a point" tool to probe specific locations (remember to set the correct Rel E for the probe point).
